# 🍽️ Master Chef Restaurant Website

A modern and responsive restaurant landing page created using **HTML5**, **CSS3**, and **Media Queries**.  
This project is designed to showcase a restaurant website UI with multiple sections like food categories, chef introduction, customer reviews, newsletter form, and responsive navigation.

The website focuses on creating a clean, attractive, and user-friendly design experience across different screen sizes.  
Built completely with frontend technologies without using any framework.

---

## Live Features

- Responsive Navigation Bar
- Mobile Menu using Checkbox Hack
- Hero Section with Call-to-Action Buttons
- Food Categories Cards
- Booking Availability Form
- COVID-19 Awareness Section
- Restaurant Story Section
- Ingredients Showcase
- Chef Introduction Section
- Customer Testimonials
- Newsletter Contact Form
- Footer with Social Media Icons
- Fully Responsive Layout using Media Queries
